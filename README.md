[ABOUT]
This tool helps you clean cache files for FiveM, Discord, and Windows 
to fix FPS drops, texture bugs, and connection issues.

[HOW TO USE]
1. Right-click the .bat file and select "Run as Administrator" (Or just double-click).
2. Type 'Y' to confirm cleaning.
3. Wait for the process to finish.

[DISCLAIMER]
Use at your own risk. This script only deletes temporary cache files 
and does not touch your personal data or game files.
